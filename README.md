# Demographic Insights from Aadhaar Enrolment Data – India (2025)

This project analyzes Aadhaar enrolment data to understand India’s demographic patterns across age groups and states using Power BI and SQL.

## Objective  
To identify age-wise and region-wise enrolment trends and derive meaningful population insights for policy, planning, and analysis.

## Key Insights  
- Adults (18+) form around 50% of the population  
- Youth (5–17) represent a strong future workforce  
- Children (0–5) are a smaller segment  
- Uttar Pradesh, Maharashtra, and Bihar have the highest adult enrolments  
- Urban districts show higher enrolment than rural areas  
- Population patterns reflect migration and employment trends  

## Tools Used  
- Power BI  
- SQL (PostgreSQL)  
- Data Modeling  
- Data Visualization
